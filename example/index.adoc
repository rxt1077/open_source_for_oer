== Fork me on GitHub!

You can find a https://github.com/rxt1077/open_source_for_oer[git repo for this text here].
Make sure you have https://git-scm.com/download/[git installed] and then https://docs.github.com/en/get-started/quickstart/contributing-to-projects[fork this repo and create a local clone].
